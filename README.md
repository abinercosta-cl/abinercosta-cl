## 🚀 Projetos em Destaque

### 🏪 [EstoquePRO](https://github.com/abinercosta-cl/EstoquePRO)
> Sistema POS e gestão de estoque para negócios híbridos (produtos + serviços)

- **Stack:** Next.js 16 · TypeScript · Prisma ORM · SQLite · Tailwind CSS
- **Features:** PDV com checkout seguro, controle de estoque, relatórios em Excel, autenticação JWT com HTTP-Only Cookies, RBAC server-side
- **Status:** 🟡 Em desenvolvimento ativo
